Static maskString

  • maskString(value: string, maskChar?: string, leaveLast?: number): string
    • Defined in data.ts:42

    Masks the specified string. For eaxmple to mask a phone number but leave the last 4 digits visible you could use maskString(phone, "X", 4).

    Parameters

    • value: string

      The original value to be masked.

    • Optional maskChar: string

      Optional character to be used on the masking, default is *.

    • Optional leaveLast: number

      Optional, leave last X positions of the string unmasked, default is 0.

    Returns string

    The masked string.
